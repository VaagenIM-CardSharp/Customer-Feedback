# Bakgrunn for prosjektet

En måte for at brukere kan gi tilbakemelding om bedriften og hva de synes samt om hva som kan forbedres. Et veldig simplet stjerne system uten komplekse mekanismer som alle kjenner til og kan lett bruke, tilgjengelig for alle. Vedlagt et valgfri kommentarfelt nedenfor hvor brukere kan skrive mer detaljert og spesifisere vurderingen dems. Alle vurderingene blir lagret i en Json database som kan lett hentes av andre utviklere.

-------------------------------

# Pensum

### Driftstøtte
- Forenkle og automatisere arbeidsprosesser i utvikling av IT-løsninger for å øke effektivitet.
- Planlegge og dokumentere arbeidsprosesser og IT-løsninger ved Mkdocs og Github Repos.
- Planlegge, dirfte og implementere IT-løsninger som ivaretar.informasjonssikkerhet og gjeldende regelverk for personvern. F.eks, ikke samle inn personopplysninger til kunder uten dems tillatelse.

### Brukerstøtte
- Utøve brukerstøtte og veilede i relevant programvare.
- Kartlegge behovet for og utvikle veiledninger for brukere og kunde ved hjelp av dette prosjektet med lærer Karin.
- Bruke og tilpasse kommunikasjonsform og fagterminilogi i møte med brukere og kunder som gjør det mer forstålig.
- Feilsøke og rette feil ved hjelp av feilsøkingsstrategier og relevante rammeverk som AI, IDE feilsøking og Google søk.

### Utvikling
- Vurdere fordeler og ulemper ved ulike programmeringsspråk og velge anvende relevante programmeringsspråk og algoritmer i eget arbeid. F.eks, bruke CSS og HTML til lette nettsider ved hjelp av Python som samler inn all dataen i stedet for mer komplekse språk.
- Vurdere brukergrensesnitt til IT-tjenester som er tilåasset brukerens behov.
- Beskrive ulike datalargringsmodeller og metoder for å hente ut og sette inn bestemte data fra databaser som brukes av andre systemer.
