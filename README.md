# A simple customer feedback system that lets customers rate their experience

Can be implemented anywhere and stores all received data in a Json file with Google Analytics as a third party for further assesments about the users visiting and using the program.  

## Documentation
Documentation is built in with the code, and for more information about the project, visit the Mkdocs file by writing `mkdocs serve` in the terminal. Click the link given and you shall receive a website with the following information: competence goals and background for project.

## How to use Google Analytics
Insert webpage link into a new Google Analytics project and insert the Google Analytics key as ID into the link on the bottom section in `feedback.html` to connect it.
